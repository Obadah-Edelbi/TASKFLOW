# TaskFlow 🚀

Modern task management web application built with Angular.

## ✨ Features

- Authentication System
- Admin Dashboard
- Task Management
- Dark / Light Mode
- Profile & Settings Pages
- Responsive Design
- Task Status Management
- Clean Modern UI

## 🛠 Technologies Used

- Angular 17
- TypeScript
- RxJS
- Angular Material
- Tailwind CSS
- Node.js
